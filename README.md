# 🚀 AI Practice Hub – Deep Learning & Machine Learning Projects

> A beginner-friendly collection of **real-world AI projects** with clean UI, model training, and hands-on notebooks.

---

## 🌟 Why this repository?

Most students struggle with:

* ❌ Understanding how ML/DL works in real projects
* ❌ Finding simple, working implementations
* ❌ Connecting theory with practical code

👉 This repository solves that by providing **ready-to-run projects with explanations**.

---

## 📂 Projects Included

### 🔢 Digit Recognition

* Handwritten digit classification using deep learning
* Covers preprocessing, training, and evaluation

### 🐶🐱 Dogs vs Cats Classification

* Image classification using CNN
* Real-world dataset handling

### 🌸 Flower Recognition

* Multi-class classification
* Feature extraction and model tuning

### 🎯 Object Detection

* Detect objects in images
* Introduction to bounding boxes & detection concepts

### 😊 Sentiment Analysis

* NLP-based sentiment classification
* Text preprocessing and prediction

## 🚀 How to run

```bash
git clone https://github.com/DarshiniMahesh/AI-Practice.git
cd your-repo-name
pip install -r requirements.txt
```

Open notebooks using Jupyter Notebook or VS Code.
