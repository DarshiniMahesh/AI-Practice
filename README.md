# 🤖 AI Practice

A collection of hands-on Artificial Intelligence and Machine Learning projects built using Python and Google Colab. This repository focuses on solving real-world problems using deep learning and computer vision techniques.

---

## 🚀 Projects Included

### 1. 🔢 Digit Recognition

* Built a model to recognize handwritten digits.
* Dataset: MNIST
* Techniques: CNN (Convolutional Neural Networks)

---

### 2. 🐶🐱 Dogs vs Cats Classification

* Image classification model to distinguish between dogs and cats.
* Techniques: Deep Learning (CNN), Image Preprocessing

---

### 3. 🌸 Flower Recognition

* Classifies different types of flowers from images.
* Focus on multi-class classification problems.

---

### 4. 🗑️ Garbage Classification

* Categorizes waste into different classes.
* Real-world application: Smart waste management systems.

---

### 5. 🏠 Interior Designer AI

* AI-based model for interior design suggestions (or classification).
* Focus on practical AI application.

---
